*** Create a Maven project from template "maven-archetyoe-quickstart".

mvn archetype:generate -DgroupId=iscope -DartifactId=iscope -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false


*** Create a maven rest API project using "maven-archetype-webapp"

mvn archetype:generate -DarchetypeArtifactId=jersey-quickstart-grizzly2 -DarchetypeGroupId=org.glassfish.jersey.archetypes -DinteractiveMode=false -DgroupId=com.iscope.rest -DartifactId=eapi -Dpackage=com.iscope.rest -DarchetypeVersion=2.17


Then in Eclipse File->Import it into eclipse.


*** run app created in java project

java -cp target/iscope-1.0-SNAPSHOT.jar iscope.App


*** mongodb archetype ...

mvn archetype:generate -DgroupId=com.iscope.mongodb -DartifactId=entitydb -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false 


*** web archetype ...

mvn archetype:generate -DgroupId=com.iscope.web -DartifactId=iscopeapi -DarchetypeArtifactId=maven-archetype-webapp -DinteractiveMode=false


*** Mongodb as a maven dependency

Put the following dependency into your POM to use mongoDB.

<dependency> 
	<groupId>org.mongodb</groupId> 
	<artifactId>mongo-java-driver</artifactId> 
	<version>2.11.0</version> 
</dependency> 

- See more at: http://www.wegtam.net/article/use-mongodb-your-java-maven-project#sthash.9RuVmyCw.dpuf
